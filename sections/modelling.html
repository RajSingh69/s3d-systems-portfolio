Architectural Visualisation

Gurdwara Library Project

A realistic render concept for the newly built first Sikh temple in Surrey. The library room had not yet been built physically, so the entire environment was recreated digitally from scratch using real floor plans and measurements.

The goal was to visualise atmosphere, layout, spacing and flow before physical construction began, creating a realistic preview of the future space.
